Steps to run the project
_________________________

1- install meavn.

2- open server project (I'm using vscode)

3- open the terminal and run these commands

-mvn clean
-mvn validate
-mvn compile

4- open Server.java and run (ctrl + f5 or any other way).

5- open Client project 

6- open the terminal and run these commands

-mvn clean
-mvn validate
-mvn compile

7- open Client.java and run (ctrl + f5 or any other way).
